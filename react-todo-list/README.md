# ✅ React To Do List

Projeto desenvolvido utilizando **React**, com foco na criação de uma aplicação para gerenciamento de tarefas, permitindo organizar atividades de estudo por meio de uma interface moderna, responsiva e intuitiva.

## 🌐 Projeto Online

👉 **Acesse o projeto:** https://projeto-react-to-do-list.vercel.app/

## 📋 Funcionalidades

- Adição de novas tarefas
- Edição e exclusão de tarefas
- Organização das tarefas em **"Para estudar"** e **"Concluído"**
- Marcação de tarefas como concluídas
- Feedback visual para lista vazia (_Empty State_)
- Modal para criação e edição de tarefas
- Persistência dos dados utilizando **LocalStorage**
- Interface responsiva e animações na lista de tarefas

## 🛠️ Tecnologias utilizadas

- React
- JavaScript
- Context API
- CSS Modules
- LocalStorage
- Vite

## 💻 Sobre o projeto

Este projeto foi desenvolvido como parte dos meus estudos em React, permitindo praticar conceitos fundamentais como componentização, gerenciamento de estado, compartilhamento de dados entre componentes utilizando Context API e persistência de informações no navegador através do LocalStorage.

## 🚀 Aprendizados

Durante o desenvolvimento deste projeto, pratiquei:

- Componentização com React;
- Gerenciamento de estado utilizando `useState` e `useEffect`;
- Compartilhamento de estado global com **Context API**;
- Manipulação de formulários controlados;
- Persistência de dados utilizando **LocalStorage**;
- Organização de estilos com **CSS Modules**;
- Estruturação de projetos React utilizando **Vite**;
- Boas práticas de organização de código.

## ▶️ Como executar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/gabriel-pereira0/Projeto_React_To_Do_List
```

2. Acesse a pasta do projeto:

```bash
cd react-todo-list
```

3. Instale as dependências:

```bash
npm install
```

4. Execute o projeto:

```bash
npm run dev
```

5. Abra no navegador:

```text
http://localhost:5173
```
